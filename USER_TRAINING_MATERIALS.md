## 🎓 1. Administrator Training Package

### ⚡ **Quick Start Guide** (5 Minutes)

#### **Step 1: System Login**
```
1. Open browser and navigate to your school's URL
2. Enter your email and password
3. Click "Login" to access the dashboard
4. Verify your role shows as "Administrator"
```

#### **Step 2: Dashboard Overview**
```
- Student Count: Total enrolled students
- Teacher Count: Active teaching staff
- Financial Summary: Revenue and expenses
- Recent Activities: Latest system activities
```

#### **Step 3: Essential First Tasks**
```
1. Update School Profile (Settings → School Setup)
2. Configure Academic Year (Academic → Academic Year)
3. Create Classes and Sections (Academic → Classes)
4. Add Teachers (Users → Teachers)
5. Set Fee Structure (Finance → Fee Setup)
```

### 🔌 **API Documentation**

#### **Authentication**
```javascript
// Login endpoint
POST /api/auth/login
{
  "email": "user@school.com",
  "password": "password123"
}

// Response
{
  "token": "jwt_token_here",
  "user": {
    "id": 1,
    "name": "John Doe",
    "role": "teacher"
  }
}
```

#### **Student Management**
```javascript
// Get students
GET /api/students?class_id=1&section=A

// Create student
POST /api/students
{
  "name": "Jane Smith",
  "class_id": 1,
  "section": "A",
  "parent_email": "parent@email.com"
}
```

#### **Academic Operations**
```javascript
// Mark attendance
POST /api/attendance
{
  "class_id": 1,
  "date": "2025-01-03",
  "students": [
    {"student_id": 1, "status": "present"},
    {"student_id": 2, "status": "absent"}
  ]
}

// Enter grades
POST /api/grades
{
  "student_id": 1,
  "subject_id": 1,
  "assessment_type": "CA1",
  "score": 85
}
```
