## 🎯 Quick Selection Guide

### By Task Type

**Building Features:**
- Frontend work → **Frontend Expert** or **UI/UX Expert**
- Backend APIs → **Backend Expert**
- Database changes → **DBA Expert**
- Financial logic → **Finance Expert** or **Accounting Expert**

**Fixing Issues:**
- UI bugs → **Frontend Expert** or **UI/UX Expert**
- API errors → **Backend Expert**
- Database problems → **DBA Expert**
- Accounting errors → **Finance Expert** or **Accounting Expert**
- Test failures → **QA Expert**

**System Operations:**
- Deployment → **DevOps Expert**
- Performance → **Backend Expert** or **DBA Expert**
- Security → **Backend Expert** or **QA Expert**
- Monitoring → **DevOps Expert**

**Planning & Coordination:**
- Feature planning → **Project Manager**
- Risk assessment → **Project Manager**
- Documentation → **Project Manager**
- Team coordination → **Project Manager**

## 📞 How to Invoke Agents

### For Claude/Gemini:
```
"Backend Expert: Optimize the payment processing API endpoint"
"Finance Expert: Review journal entry logic for compliance"
"DBA Expert: Create stored procedure for attendance calculation"
"Frontend Expert: Add validation to student enrollment form"
```

### For OpenCode:
```
@Backend-Expert fix the slow query in payments controller
@DBA-Expert add index to payment_entries table
@Frontend-Expert update the dashboard layout
@QA-Expert test the new billing feature
```

## 📋 Agent Collaboration Patterns

### Feature Development Flow:
1. **Project Manager** → Plan and break down requirements
2. **DBA Expert** → Design database schema
3. **Backend Expert** → Implement API endpoints
4. **Frontend Expert** → Build UI components
5. **Finance Expert** → Validate accounting logic
6. **QA Expert** → Test implementation
7. **DevOps Expert** → Deploy to production

### Bug Fix Flow:
1. **QA Expert** → Reproduce and document bug
2. **Relevant Expert** → Implement fix
3. **QA Expert** → Verify fix
4. **Project Manager** → Document resolution

### Database Change Flow:
1. **DBA Expert** → Schema design
2. **Backend Expert** → Update ORM models
3. **Finance Expert** → Review financial impact
4. **QA Expert** → Test data integrity
5. **DevOps Expert** → Execute migration
