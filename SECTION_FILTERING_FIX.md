Added Section Filtering Logic**\n\n```javascript\n// Filter sections to only show those that have classes in the school\nconst getAvailableSections = () => {\n  return sectionsList.filter(section => {\n    // Check if there are any classes for this section\n    const hasClasses = classes?.some((item: ClassItem) => {\n      const sectionName = (typeof item.section === 'object' && item.section) \n        ? item.section.section_name \n        : item.section;\n      return sectionName === section.section_name;\n    });\n    return hasClasses;\n  });\n};\n```\n\n### **2. Updated Modal to Use Filtered Sections**\n\n**Before**: Modal showed all sections from API\n```javascript\ndataSource={sectionsList}  // Showed all sections\n```\n\n**After**: Modal only shows sections that have classes\n```javascript\ndataSource={availableSections}  // Only shows sections with classes\n```\n\n### **3.
